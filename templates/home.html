Gaming With Hand Gestures

In This project we have developed a technique where we can be able to play the subway surfers game using our hands

The instructions to play the game are as follows:

  • The link to the game is provided below click on that link
  • After the game is loaded click on "PRESS TO PLAY"
  • If you want the character to move UP show ONE finger
  • If you want the character to move DOWN show TWO fingers
  • If you want the character to move RIGHT show THREE fingers
  • If you want the character to move LEFT show FOUR fingers
